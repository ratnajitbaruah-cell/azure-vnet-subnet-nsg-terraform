# Azure VNet, Subnet & NSG Deployment using Terraform

## Overview
This project demonstrates Infrastructure as Code (IaC) using Terraform to deploy Azure networking resources.

## Resources Created
- Azure Resource Group
- Azure Virtual Network (VNet)
- Azure Subnet
- Azure Network Security Group (NSG)

## Technologies Used
- Microsoft Azure
- Terraform

## Commands Used

terraform init

terraform validate

terraform plan

terraform apply

terraform destroy
